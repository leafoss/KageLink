# Instruções de atualização — KageLink 3.3.0

## APK

1. Feche o workspace antigo `_build_workspace` se estiver aberto.
2. Execute `COMPILAR_APK.bat`.
3. Instale `KageLink-v3.3.0.apk` sobre a versão atual.
4. Use o mesmo perfil/endereço/token já salvo.

## PC Agent

1. Feche o KageLink no Windows.
2. Execute `installer\CRIAR_INSTALADOR.bat` no computador de compilação.
3. Instale `installer\output\KageLink-PC-Agent-Setup-v3.3.0.exe` sobre a versão anterior.
4. O instalador mantém dados de usuário e conexão.
5. Abra o jogo e então o KageLink.

O APK 3.3.0 necessita do PC Agent 3.3.0 para a aba GAME. O chat permanece compatível com o protocolo anterior, mas os novos endpoints não existem em agentes antigos.
