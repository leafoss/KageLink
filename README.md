# KageLink

KageLink is a companion application for Shinobi Story Online that connects the desktop client and mobile interface.

## Features

- Remote chat (OOC / IC)
- Desktop Agent
- Android application
- Installer
- Cloudflare tunnel support

## Repository Structure

- `android_overlay/` – Android application
- `pc_agent/` – Desktop agent
- `installer/` – Windows installer
- `assets/` – Images and assets
- `docs/` – Documentation
